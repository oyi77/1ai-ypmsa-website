Pertanyaan Umum

Hal-hal yang sering ditanyakan tentang program Sedekah Sehat YPSMA.

Apa itu program Sedekah Sehat YPSMA?

Program kebersihan dan pengelolaan sampah terpadu untuk PPTQ Darussalam Jombang. Dengan target Rp 40 juta, kami akan membangun Tempat Pembuangan Sementara (TPS) terpilah, mengadakan 50 tong sampah untuk setiap kelas dan asrama, serta membeli mesin pencacah untuk produksi kompos.

Kapan target penggalangan dana ini selesai?

Kami menargetkan dana terkumpul dalam 3 bulan ke depan. Setelah dana mencukupi, pembangunan TPS dan pengadaan fasilitas akan dilakukan secara bertahap dengan prioritas pada area yang paling membutuhkan.

Apakah saya bisa berdonasi rutin setiap bulan?

Tentu! Anda bisa menjadi donatur tetap Sedekah Sehat dengan nominal Rp 25.000/bulan atau sesuai kemampuan. Kami akan mengirimkan laporan perkembangan program secara berkala via WhatsApp agar Anda tahu dampak donasi Anda.

Bagaimana cara transfer donasi?

Transfer ke rekening YPSMA di BRI atau BSI atas nama Yayasan Pendidikan dan Sosial Mamba'ul Maarif. Setelah transfer, konfirmasi bukti pembayaran ke WhatsApp 0822-3455-1160 untuk pencatatan dan didoakan oleh para santri.

Apakah donasi saya bisa digunakan untuk program lain?

Donasi yang masuk khusus untuk program Sedekah Sehat akan dialokasikan 85% untuk kebutuhan fisik (TPS, tong sampah, mesin kompos) dan 15% untuk biaya operasional serta publikasi program. Jika ada kelebihan dana, akan digunakan untuk perawatan dan pengembangan program kebersihan pondok.
